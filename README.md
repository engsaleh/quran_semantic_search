# Quran Semantic Search 
محرك بحث دلالي للقرآن الكريم يعتمد على النماذج اللغوية لفهم المعنى لا الألفاظ.

## فكرة المشروع
يتيح البحث في الآيات حسب المعنى، فيُظهر نتائج مرتبطة بالفكرة حتى إن لم تتطابق الكلمات

مثلاً: عند البحث عن آيات عن جزاء الظالمين، يعرض الآيات ذات الصلة بالمعنى نفسه فيعتبر عذاب مرادف لجزاء والظالمين مرادف للمجرمين

## المكونات الرئيسة
- **FastAPI**  لبناء واجهة REST سريعة.  
- **SentenceTransformers**  لتوليد التضمينات الدلالية.  
- **intfloat/multilingual-e5-base**  نموذج متعدد اللغات يفهم العربية.  
- **FAISS**  للبحث السريع في فضاء التضمينات.  
- **rank-bm25**  خوارزمية البحث اللفظي التقليدي.  
- **HTML / CSS / JS**  واجهة أمامية بسيطة لعرض النتائج.

## التشغيل
1. استنساخ المشروع:
   ```bash
   git clone https://github.com/USERNAME/quran-semantic-search.git
   cd quran-semantic-search/backend
   ```
2. تثبيت المتطلبات:
   ```bash
   pip install -r requirements.txt
   ```
3. تشغيل الخادم (الواجهة الخلفية):
   ```bash
   uvicorn main:app --reload
   ```
4. فتح ملف الواجهة الأمامية:
   ```
   frontend/index.html
   ```
